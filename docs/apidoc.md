## totalAgent

### Description

Returns the sum of input values

### Samples

```json
[
  {
    "inputs": [
      {
        "a": 1
      },
      {
        "a": 2
      },
      {
        "a": 3
      }
    ],
    "params": {},
    "result": {
      "a": 6
    }
  },
  {
    "inputs": [
      [
        {
          "a": 1,
          "b": -1
        },
        {
          "c": 10
        }
      ],
      [
        {
          "a": 2,
          "b": -1
        }
      ],
      [
        {
          "a": 3,
          "b": -2
        },
        {
          "d": -10
        }
      ]
    ],
    "params": {},
    "result": {
      "a": 6,
      "b": -4,
      "c": 10,
      "d": -10
    }
  }
]
```
